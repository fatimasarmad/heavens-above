getTable

Fetches data from the IridiumFlares.aspx page, parses it, and stores it in a local database.

getTable(config: Object): void
Parameters
config (Object) Configuration object.
Name Description
config.database Array The existing database to which new data will be added. Defaults to an empty array.
config.counter number The current count of pages fetched. Defaults to 0.
config.opt number The options for the request. Defaults to 0.
config.root string The root directory where the fetched data will be stored.
Returns
void:
